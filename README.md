# socket-chat-room
Node.js로 채팅방 만들기 

## 목적

- 웹 소켓을 이해하고 실제로 채팅방을 구현해 보자!!

## 기능

- 채팅방 생성(비밀번호/ 방인원 배정/ 역할 등)
- 채팅방 조회
- 채팅방 입장
- 채팅 
- 이미지 업로드

## npm

- npm i ws
- npm i socket.io
- npm i mongoose
